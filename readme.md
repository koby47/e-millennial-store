# The E-Millennial Store

## Overview

The E-Millennial Store is a responsive e-commerce web application developed as a frontend capstone project using HTML, CSS, and JavaScript. The application allows users to browse products, manage a shopping cart, validate customer information, and complete payments through Paystack.

## Project Objectives

* Build a responsive e-commerce storefront.
* Implement dynamic product rendering using JavaScript.
* Create a functional shopping cart system.
* Validate customer information before checkout.
* Integrate Paystack payment processing.
* Deliver a complete purchase workflow from product selection to payment confirmation.

## Features

### Product Catalog

* Dynamic product rendering from JavaScript data.
* Responsive product grid layout.
* Product price overlay on hover.
* Mobile-friendly design.

### Shopping Cart

* Add products to cart.
* Remove products from cart.
* Increase and decrease item quantities.
* Dynamic cart badge updates.
* Automatic total calculation.

### Customer Information

* Name validation.
* Email validation.
* Phone number validation.
* Real-time validation feedback.

### Payment Integration

* Paystack payment gateway integration.
* Ghana Cedi (GHS) support.
* Customer metadata submission.
* Payment confirmation workflow.

### Order Summary

* Successful payment confirmation.
* Purchase summary display.
* Cart and form reset after completion.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Paystack Payment Gateway
